Type Diameters / Tipe Diabetes Tes

Type Diameters adalah sebuah webaite dengan teknologi Machine Learning (ML) untuk tes Tipe Diabetes. Website ini menggunakan algoritma Random Forest yang digunakan untuk memprediksi tipe diabetes.

Instruksi Penggunaan
Langkah 1

Tekan tombol Coba Test atau scroll kebawah sampai halaman Tes Tipe Diabetes.

Langkah 2

Sebelum mengisi pastikan anda Baca terlebih dahulu detail pada tombol Baca Detail yang berwarna kuning.

Langkah 3

isi form dengan benar. Kemudian tunggu beberapa saat sistem akan memprediksi hasil dari Diagnosa.

Tes Tipe Diabetes

Sebelum mengisi tes tipe diabetes harap membaca keterangan info detail untuk melihat keterangan pada setiap inputan.

Detail atribut tiap Inputan
Berikut detail keterangan setiap atribut / variabel yang ada pada inputan.
No Atribut Tipe Data Keterangan & Nilai
1. Age Integer / Bilangan Bulat Umur Pasien
2. BS Fast Float / Pecahan Gula darah saat berpuasa sebelum makan (mmol/L)
3. BS PP Float / Pecahan Gula darah dalam 90 menit setelah makan (mmol/L)
4. Plasma R Float / Pecahan Tes glukosa plasma dilakukan secara acak setiap saat (mmol/L)
5. Plasma F Float / Pecahan Tes glukosa plasma saat berpuasa biasanya dilakukan ketika pagi hari (mmol/L)
6. HbA1c Integer / Bilangan Bulat Tes ini dilakukan untuk mengukur rata-rata kadar glukosa selama 2-3 bulan terakhir (mmol/mol).
